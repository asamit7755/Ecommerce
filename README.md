## 🛠 Tech Stack

### 🔷 Frontend
- ASP.NET Web Forms (.aspx)
- Bootstrap 5.3
- jQuery 3.6
- Toastr for notifications

### 🔶 Backend
- C# (.NET Framework 4.8)
- ADO.NET for DB access
- Forms Authentication
- Session-based cart system

### 🧩 Database
- SQL Server (2019+)
- Tables: Products, Users, Orders, Categories
